PML - PhiraMural 视频背景插件 v1.0 预览版
==========================================

【快速开始】

方法一（推荐）：双击 "PML_PhiraMural.exe" 启动程序
  - 独立可执行文件，无需安装Python
  - 双击即可运行

方法二：双击 "备用启动.bat" 启动程序（备选方案）
  - 自动检测exe文件，如果没有则使用Python运行
  - 需要安装Python 3.12+

方法三：双击 "静默启动.vbs" 启动程序（无窗口）
  - 使用Python运行，不显示命令行窗口
  - 需要安装Python 3.12+

【系统要求】

- Windows 10/11
- 不需要安装Python（已有exe文件）

【关于Python环境】

本插件已打包为独立可执行文件（PML_PhiraMural.exe），无需安装Python即可运行。

如果exe文件无法运行，可以使用"备用启动.bat"，它会检测Python环境并尝试运行源代码版本。

【手动安装Python】

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.12 或更高版本
3. 运行安装程序
4. **重要**：安装时勾选 "Add Python to PATH"
5. 完成安装后重启电脑

【使用步骤】

1. 启动程序后，选择谱子文件夹或压缩包（.pez/.zip）
2. 选择视频背景文件（支持MP4/MKV/AVI/WEBM格式）
3. 根据需要设置选项：
   - 复制模式：勾选后会复制谱子到新位置再修改
   - 压缩输出：将结果打包为.zip或.pez格式（仅文件夹输入时可用）
   - 统一重命名：将音频、图片、谱子JSON改为同名（文件夹和压缩包输入都可用）
4. 选择输出位置（如果启用了复制模式，会自动同步）
5. 点击"生成"按钮开始处理

【功能说明】

- 自动验证RPE格式
- 自动剥离视频音轨（需要FFmpeg，可选）
- 清洗RPE v170不兼容字段
- 生成Phira所需的info.yml和extra.json
- 支持文件夹和压缩包两种输入方式
- 支持统一重命名文件
- 支持压缩输出为.zip或.pez格式

【FFmpeg（可选）】

FFmpeg用于剥离视频音轨，使Phira只播放无声视频。

安装方法：
1. 访问 https://ffmpeg.org/download.html
2. 下载Windows版本
3. 解压到任意目录
4. 将ffmpeg.exe所在目录添加到系统PATH环境变量

如果不安装FFmpeg，程序会直接复制原视频（带声音）。

【常见问题】

Q: 启动时提示"未找到Python环境"怎么办？
A: 请安装Python 3.12或更高版本，并确保安装时勾选了"Add Python to PATH"。

Q: 处理后的视频没有声音？
A: 这是正常现象。程序会自动剥离视频音轨，Phira只需要无声视频作为背景。

Q: 支持哪些视频格式？
A: 推荐使用MP4（H.264编码）。也支持MKV、AVI、WEBM格式。

Q: 处理后的文件在哪里？
A: 如果启用了复制模式，文件在复制目标位置；否则在原位置或指定的输出位置。
